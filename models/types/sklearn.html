# Scikit-Learn Model

This is a base model that takes in a scikit-learn model as an argument.

The models that are already integrated into this library include:

  • Adaboost
  • Elastic Net
  • Linear Regression
  • Logistic Regression
  • Random Forest
  • SGD
  • SVM
